# nodejs-web-hello-world

## Description
A POC project for nodejs.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
http://localhost:8000

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
